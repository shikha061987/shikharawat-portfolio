# Shikharawat Portfolio

This is my personal portfolio website showcasing my skills, projects, and experience as a frontend developer.  
It is built with **Angular 17** and **Tailwind CSS**, designed to be **responsive**, **modern**, and **user-friendly**.

---

## Features

- **About Me** section with brief introduction  
- **Skills** section highlighting technologies I work with  
- **Projects** section showcasing my work  
- **Contact** section with email, phone, and social media links  
- Fully **responsive design** for mobile and desktop  
- Smooth animations and interactive UI components  

---

## Technologies Used

- **Angular 17**  
- **TypeScript**  
- **Tailwind CSS**  
- **HTML & CSS**  
- **JavaScript**
